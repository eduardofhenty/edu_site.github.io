Lista exemplificativa de serviços disponíveis
  1. Obtenção de dados meteorológicos (DAEE, ANA, reanálise, etc).
  2. Organização de dados meteorológicos e outros tipos de dados.
  3. Cálculos estatísticos.
  4. Filtragem de dados.
  5. Elaboração de mapas em R, Python ou Qgis.
  6. Detecção de pontos de ruptura.
  7. Interpolação de dados 1D e 2D.
  8. Tradução de código fonte (do Fortran para R, por exemplo).
  9. Delimitação de área de drenagem.
  10. Cálculos de índice climatológicos.
  11. Cálculo de Evapotranspiração.
  12. Cálculo de normais climatológicas.
  13. Gráficos de diferentes tipo (histograma, séries temporais, ciclo diurno).
  14. Ajuste de modelos.
  15. Aplicação de métodos de machine learning.
  16. Desenvolvimento de site (esse aqui e também https://eduardofhenty.github.io/pet_galeria.github.io/index.html)
  17. Elaboração de Dashboard.
  18. Relatórios em Rmarkdown.
